■Gitの基礎
- Gitの基礎
 master : デフォルトの開発ブランチ
 origin : デフォルトの上流ブランチ
 HEAD   : 現在のブランチ
 HEAD^  : HEADの親
 HEAD-4 : HEADの祖父母の祖父母








公開


ローカルの変更をすべてコミットする

git commit -a

他の開発者向けにパッチを用意する

git format-patch origin

変更をoriginに反映する

git push

バージョンやマイルストーンのタグを付ける

git tag v1.0






マージの競合を解消


マージの競合を確認する

get diff
get diff -base $file
get diff -ours $file
get diff -theirs $file
